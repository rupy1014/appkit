# 맥비 기획자 템플릿 구독 서비스 기획

## 1. 서비스 개요

### 배경
- **맥비**: 국내 최대 기획자 모임 (10년 운영, 약 1만명 오픈채팅 커뮤니티)
- **보유 자산**: 1,000개 이상의 기획 자료 및 템플릿
- **자료 특성**: 다양한 포맷, 카테고리별 정리 중, 지속 갱신

### 서비스 컨셉
> **"기획자를 위한 지식창고"**
> - 주니어 기획자가 언제든 필요한 템플릿을 찾아 쓸 수 있는 자료 라이브러리

### 타겟 사용자
- 취업 준비 중인 예비 기획자
- 1~3년차 주니어 PM/기획자
- 실무 템플릿이 필요한 스타트업 담당자

---

## 2. 핵심 과제

### 문제 정의
| 과제 | 설명 |
|------|------|
| 낮은 지불 능력 | 주니어 대상, 유료 전환 의지 낮음 |
| 무료 익숙함 | 기존 1만명이 무료 커뮤니티에 익숙 |
| 경쟁 콘텐츠 | 무료 자료가 이미 시장에 넘침 |

### 해결 방향
- **저가 구독**: 부담 없는 가격으로 진입 장벽 낮춤
- **양으로 승부**: 1,000개 이상 템플릿의 압도적 볼륨
- **편의성**: 잘 정리된 카테고리, 쉬운 검색

---

## 3. 서비스 구조

### 플랫폼
**노션(Notion) 기반 자료 라이브러리**

선택 이유:
- 빠른 구축 가능 (2주 내 MVP)
- 검색/필터링 기능 내장
- 다양한 포맷 임베드 가능
- 구독자별 접근 권한 관리 용이

### 자료 카테고리 (예시)

```
📁 기획 템플릿 라이브러리
├── 📂 서비스 기획
│   ├── PRD (제품 요구사항 정의서)
│   ├── 화면 설계서
│   ├── 유저 플로우
│   └── 기능 명세서
├── 📂 사업 기획
│   ├── 사업계획서
│   ├── 수익 모델 캔버스
│   └── 시장 분석 템플릿
├── 📂 프로젝트 관리
│   ├── WBS
│   ├── 일정 관리
│   └── 회의록
├── 📂 UX/리서치
│   ├── 사용자 인터뷰
│   ├── 퍼소나
│   └── 여정 지도
├── 📂 데이터 분석
│   ├── 지표 정의서
│   ├── A/B 테스트 설계
│   └── 리포트 템플릿
└── 📂 취업/이직
    ├── 포트폴리오 템플릿
    ├── 이력서 가이드
    └── 면접 질문 모음
```

## 4. 가격 정책

### 단일 티어 (심플하게 시작)

| 항목 | 내용 |
|------|------|
| **가격** | 월 4,900원 |
| **제공 내용** | 전체 템플릿 라이브러리 무제한 접근 |
| **결제 방식** | 월 구독 (자동 갱신) |

### 가격 책정 근거
- 주니어 대상 → 월 5,000원 미만 심리적 저항선
- 커피 한 잔 가격 → "이 정도는 괜찮지" 인식
- 경쟁사 대비 저가 → 클래스101 월 1.9만원, 인프런 월 2.4만원

### 예상 수익
| 시나리오 | 구독자 수 | 월 매출 |
|----------|-----------|---------|
| 보수적 (1%) | 100명 | 49만원 |
| 기본 (2%) | 200명 | 98만원 |
| 낙관적 (3%) | 300명 | 147만원 |

## 5. MVP 런칭 계획

### Phase 1: 준비 (1주차)
- [ ] 핵심 템플릿 100개 선별 및 노션 정리
- [ ] 카테고리 구조 확정
- [ ] 노션 템플릿 라이브러리 페이지 구축

### Phase 2: 결제 연동 (2주차)
- [ ] 스티비 or 포트원 결제 연동
- [ ] 구독자 전용 노션 페이지 접근 권한 설정
- [ ] 랜딩 페이지 제작 (노션 or 타입폼)

### Phase 3: 얼리버드 테스트 (3주차)
- [ ] 기존 오픈채팅에서 얼리버드 모집 공지
- [ ] 얼리버드 가격: 월 3,900원 (100명 한정)
- [ ] 피드백 수집 및 개선

### Phase 4: 정식 오픈 (4주차~)
- [ ] 정가 4,900원 전환
- [ ] 지속적 템플릿 추가 (주 5개 이상)
- [ ] 월간 뉴스레터 발송

## 7. 향후 확장 방안

### 단기 (3~6개월)
- 인기 템플릿 TOP 10 분석 → 콘텐츠 집중
- 구독자 요청 기반 신규 템플릿 제작

### 중기 (6~12개월)
- Pro 티어 추가 (월 29,000원): 1:1 피드백, 스터디 참여권
- 기업 대상 B2B 라이선스 판매

### 장기 (1년 이후)
- 기획 교육 코스 연계
- 기획자 채용 플랫폼 확장
